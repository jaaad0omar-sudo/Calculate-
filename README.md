# Simple Interest Calculator | آلة حاسبة للفائدة البسيطة

هذا المشروع عبارة عن أداة بسيطة مبنية باستخدام (ذكر لغة البرمجة المستخدمة، مثلاً: Python أو JavaScript) لحساب الفائدة البسيطة بناءً على المبلغ الأساسي، معدل الفائدة السنوي، والوقت.

## معادلة الحساب (The Formula)
تستخدم الأداة المعادلة التالية:
$A = P(1 + rt)$

حيث أن:
* **A**: المبلغ الإجمالي (Total Amount).
* **P**: المبلغ الأساسي (Principal Amount).
* **r**: معدل الفائدة السنوي (Annual Interest Rate).
* **t**: الوقت بالسنوات (Time in years).

## المميزات (Features)
* واجهة سهلة الاستخدام.
* حساب دقيق وسريع.
* إمكانية إدخال أرقام عشرية.

## كيفية التشغيل (How to run)
1. قم بتحميل الملفات أو عمل `git clone` للمستودع.
2. افتح ملف (اسم الملف الأساسي) في بيئة التطوير الخاصة بك.
3. قم بتشغيل البرنامج وادخل البيانات المطلوبة.
